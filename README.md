# CMS AI Quoting -- Frontend

Vite + React + TypeScript + Tailwind CSS v4 UI for the CMS AI Quoting app.

See the top-level [`webapp/README.md`](../README.md) for full setup
instructions (backend, environment variables, demo data).

```bash
npm install
npm run dev -- --host
```

The dev server proxies `/api/*` to `http://localhost:8000` (see
`vite.config.ts`).
